Magnify

Accessibility options to assist the disabled in computer applications is becoming increasingly important. In some places, Laws have been made to ensure that everybody will be able to access the same information, even for people with disabilities. One of these options to enlarge a portion of the screen for people with eye weaknesses. Your task is to write a simple program that magnifies text graphics.

Input

The first line of the input file MAGNIFY.IN contains 3 integers, w, h, z. Starting on the 2nd line is a w*h size text graphic, (i.e. h lines with w characters on each line.) z is the magnification factor.

Output

The output file MAGNIFY.OUT should be a wz*hz graphic, magnified from the input. Each 1*1 character in the input should be magnified to z*z characters. The sample output should demonstrate this clearly.

Constraints

1 <= w,h <= 100

1 <= z <= 10

The output file will not exceed 500K

All characters in the text graphic are alphanumeric (A-Z, a-z, 0-9)
